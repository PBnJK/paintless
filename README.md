# Paintless!

A text-based painting program for your browser, made with no CSS styling for [flavorless](https://flavorless.hackclub.com)!

## Features

- Numerous art tools! (5)
- Incredible resolution! (288x48)
- Saving!
- Jazzy background music by yours truly!
